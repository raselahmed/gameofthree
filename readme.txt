
1. Technology Spring Boot,Apache Tomcat,JSF, JS Socket lib.
2. User action Api baesd on REST service.
3. Game situation Realtime update to players by using Spring web socket. 
2. The game is designed for two players only.
3. This is maven based project.

To see the output.
1. Run the spring boot project as java application, it will deployed in Tomcat server automatically.
4. Open two different browser for two players. 
5. Type the URL in both browsers http://localhost:9090/gameofthree/index.xhtml
6. Click on 'JoinButton'
7. The click on StartGameButton. You can click from any browser, but the initial number are generated by first joined Player.
8. Then you will Real time game status update on the screen.
